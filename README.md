# SecureVote — Advanced Online Voting System

## Quick Start (No install needed)
Open frontend/index.html in browser — works instantly!

## Demo Credentials
- Admin: admin@vote.com / Admin@123
- Voter: voter@vote.com / Voter@123

## With Backend
cd backend && npm install && npm start
Then open: http://localhost:3001

## Features
- Face Recognition (camera capture + verification)
- 2-Step OTP Verification (email/SMS)
- Biometric Auth (WebAuthn/fingerprint/Face ID)
- Duplicate Vote Prevention
- Account lockout after 5 failed attempts
- JWT authentication
- Admin dashboard with live results
- Blockchain-style vote receipts
- Security audit log

## Files
- frontend/index.html  — Complete voting app
- frontend/styles.css  — All styles
- frontend/app.js      — All frontend logic
- backend/server.js    — REST API
- backend/package.json — Dependencies
